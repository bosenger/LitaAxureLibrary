输出预览文件

由于使用了字体图标，在输出HTML前，需将输出设置进行调整，增加web font。

具体位置如下：Publish-Generate HTML Files-Font-add

输入字体名称：andesign

选择Link to CSS

输入以下内容：


https://at.alicdn.com/t/font_972185_biji9rw09zc.css

在Axure中由于不具备图标字体的链接机制，所以需下载字体文件并安装在系统中才可以使用；

下载地址在GIt项目的font文件夹。

安装后，上方的所有图标应该可以正常显示，在需要使用图标时从此处复制到目标原型使用。
